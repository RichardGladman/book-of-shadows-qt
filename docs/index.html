How to Build

Quick Start

For those who know, follow these steps:

  • git clone https://github.com/RichardGladman/book-of-shadows-qt.git
  • cd book-of-shadows-qt
  • mkdir build
  • cd build
  • cmake ..
  • make
  • sudo make install
